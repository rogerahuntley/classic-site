Dog Bribes RPG Project #2 (abandoned)
So far, this game was made before any change-logs were documented. Current features are as follows:
  • Grid-based map.
  • Player Character
  • Player movement - WASD
  • Player Sprint.
  • Adjustable scale.
  • debug mode
  • Camera pans when you get close to the edge
  • Framework set up to call specific functions when player coordinates hit a specific number.
All future features and changes will be documented here.
